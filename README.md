## Setup Instructions
Clone repository and in the clone repo run the following commands.
```angular2html
composer install
php artisan serve //will start server at http://127.0.0.1:8000
```
Open the link in your browser

## Unit tests
Run unit tests

```angular2html
./vendor/bin/phpunit
```
